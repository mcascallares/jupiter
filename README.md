# Jupiter Notebooks

Jupyter notebooks to get your hands on different technologies.


## Requirements

Python 2.7


## Running

* Install Jupyter and notebook dependencies.
```
pip install -r requirements.txt
```

* Start Jupyter
```
jupyter notebook
```

* Browse http://localhost:8888 and open the desired notebook
